Через этот интерфейс создаются только рядовые сотрудники для учета 
рабочего времени и так же для указания в поле водитель.
Руководящие должности с паролем и доступом создаются только через админку - создать сотрудника
(Галочка - есть доступ появляется после того как сотрудник авторизуется на сервисе). 
Что бы удалить доступ к сайту надо убрать галочки активности через админку в поле users.
Что бы сотрудник больше не выпадал в списках сайта необходимо деактивировать сотрудника через  
web интерфейс или убрать активность через админку в поле сотрудники.
Руководитель реализация.


%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% СОЗДАНИЕ ДВИЖЕНИЯ %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Существует 4 типа перемещения:

ПЕРЕМЕЩЕНИЕ
"откуда" только свои места хранения
"куда" только свои места хранения
Поле "цена" неактивно

ОТПРАВЛЕНИЕ
"откуда" только свои места хранения
"куда" только чужие места хранения
Поле "цена" неактивно

РЕАЛИЗАЦИЯ
"откуда" все места хранения
"куда" только контрагенты
Поле "цена" активно

СПИСАНИЕ
"откуда" только свои места хранения склады
"куда" только места хранения склады, бригады и авто
"МАТЕРИАЛЫ" только ГСМ и запчасти
Поле "цена", "водитель", "транспортное средство" неактивны

Движение материалов:

1. Перемещение (между своими местами хранения)
✅ Только между местами хранения, где пользователь является создателем
❌ Нельзя перемещать с чужого склада на свой
❌ Нельзя перемещать со своего на чужой
❌ Контрагенты не участвуют
❌ Цена не указывается
✅ Выполняется сразу


2. Отправление (со своего на чужое)
✅ Отправитель: только свои места хранения
✅ Получатель: только чужие места хранения (не свои)
❌ Контрагенты не участвуют
📦 Получатель видит уведомление и подтверждает получение
❌ Цена не указывается

3. Реализация (продажа контрагенту)
✅ Отправитель: любые места хранения (свои или чужие) - здесь важно кто владелец материала
✅ Получатель: только контрагенты
💰 Обязательно указание цены
✅ Выполняется сразу
✅ Цена указывается

4. Списание
✅ Отправитель: свои склады или автомобили
❌ Нельзя списывать с чужих мест хранения
❌ Контрагенты не участвуют
✅ Выполняется сразу
❌ Цена не указывается

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% СОЗДАНИЕ ДВИЖЕНИЯ %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Заходя в интерфейс должности вы получаете временные права должности в интерфейс которой входите. 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% МЕСТА ХРАНЕНИЯ %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% 

Местами хранения являются объекты, где физически размещаются материалы (склады), куда списываются материалы 
('транспортные средства', 'бригады') а так же куда реализуются материалы ('контрагенты').
Каждое место хранения имеет владельца — пользователя (должность), который его создал и управляет остатками
материалов на нём.

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% ОСТАТКИ МАТЕРИАЛОВ %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% 

Видеть можно только остатки находящиеся на своих местах хранения.  
Остатки нельзя удалить или редактировать на прямую, только через документ  
поступление, перемещение, отправление, списание.

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% ПРИХОДОВАТЬ МАТЕРИАЛЫ %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% 

Создавать новые остатки можно только на свои места хранения.
Это гарантирует, что вы управляете только теми запасами, за которые отвечаете.

Дату поступления по умолчанию стоит "сегодня". Выставить дату можно в ручную но не более 7 дней назад,  
так же нельзя выбрать дату, которая не наступила.
Источник поступления позволяет выбрать контрагента (необязательное заполнение)
Место хранения (место хранения Склад) можно выбрать только места хранения своей должности.
Материал можно выбрать любые. 

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% ЖУРНАЛ ПОСТУПЛЕНИЙ %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% 

На каждом поступлении кнопка просмотр, кнопка редактировать (активна 5 дней после создания ПОСТУПЛЕНИЯ), 
кнопка удаления (активна 5 дней после создания). 
Поступление напрямую связанно с остатками и редактирование количества, склада поступления ведет к изминениям в остатках.
При редактировании поступления происходит проверка на остаток материала на указанном складе, 
так как указанное количество материалов может быть уже перемещено другим документом.
